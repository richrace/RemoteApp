invoke(callback, params={}) click to toggle source

This is the method the Commands call, hence the need for the Callback and params. Even though the Synchronous method doesn’t need a callback, it needs the constant from XbmcConnect (NOCALLB) to ensure it is what the developer wants.

# File app/helpers/xbmc/command.rb, line 24
def invoke(callback, params={})
  if XbmcConnect::NOCALLB == callback
    res = XbmcConnect.sync_connect(command, params)
    return res
  else
    XbmcConnect.async_connect(callback, command, params)
  end
end
