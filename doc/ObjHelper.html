Calls delete_image method and saves the object with nil pointing to the l_thumb entry.

# File app/helpers/obj_helper.rb, line 28
def destroy_thumb_image
  unless self.l_thumb.blank?
    delete_image(self.l_thumb)
    self.l_thumb = nil
    self.save
  end
end
make_sort_title(title) click to toggle source

Strips the leading ‘The/An/A’ of a given string. Returns the String without altering the original. This is not case sensitive.

# File app/helpers/obj_helper.rb, line 11
def make_sort_title(title)
  title.sub(/^(the|a|an)\s+/,'')
end
