CommandsHelper

Helper module to help with the commands. It selects the correct version of the API for the current XBMC Configuration

Public Instance Methods

clean_video_lib(callback) click to toggle source

Selects the correct XBMC version API. Only version 3/4 of the JSON RPC API is supported. Requests XBMC to clean the VideoLibrary

# File app/helpers/commands_helper.rb, line 37
def clean_video_lib(callback)
  version = XbmcConfigHelper.current_config.version.to_i
  unless version.nil?
    if (version == Api::V4::VERSION)  || (version == 3)
      Api::V4::VideoLibrary.clean(callback)
    end
  end
end

# File app/helpers/commands_helper.rb, line 23
def scan_video_lib(callback)
  version = XbmcConfigHelper.current_config.version.to_i
  unless version.nil?
    if version == Api::V2::VERSION
      Api::V2::VideoLibrary.scan(callback)
    elsif (version == Api::V4::VERSION)  || (version == 3)
      Api::V4::VideoLibrary.scan(callback)
    end
  end
end
