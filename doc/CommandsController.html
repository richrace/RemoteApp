CommandsController

Controller for the Commands that can be sent to the XBMC Server. These differ from controls as these are inputs to the UI. For example, it can move the input or scan libraries.

def change_vol_callback
  if @params['status'] == 'ok'
    cur_vol = @params['body'].with_indifferent_access[:result]
    WebView.execute_js("updateVolume('#{cur_vol}');")
  end
end

mute_callback() click to toggle source

Updates the volume with whether or not the mute toggle is active.

# File app/Commands/commands_controller.rb, line 131
def mute_callback
  if @params['status'] == 'ok'
    result = @params['body'].with_indifferent_access[:result]
    if result
      cur_vol = "Muted"
      WebView.execute_js("updateVolume('#{cur_vol}');")
    else
      load_volume
    end
  end
end
